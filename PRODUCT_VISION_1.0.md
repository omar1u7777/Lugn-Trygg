# Lugn & Trygg - Version 1.0 Product Vision

## Core User Experience

**Single Focus:** Quick daily mood check and gentle reflection support.

**Why this wins:** In a moment of stress or low energy, users need something simple that takes less than 2 minutes and provides immediate value without overwhelming choices.

## User Journey (Version 1.0)

1. **Open App** → Immediate mood check prompt
2. **Log Mood** → 3-tap interface (emoji + intensity + optional note)
3. **Gentle Reflection** → AI companion offers 1-2 supportive questions or breathing exercise
4. **Daily Pattern** → Subtle insights on mood trends (no complex analytics)

## What We Hide Initially

- Advanced analytics and charts
- Multiple relaxation techniques (keep 1-2 core ones)
- Social features
- Complex journaling
- Gamification leaderboards
- Voice features
- Premium upsells (integrate naturally later)

## Success Metrics

- Daily active users logging mood
- Session length < 3 minutes
- User retention at 7 days
- Positive feedback on simplicity

## Technical Scope

- Simplify navigation to 3-4 main items
- Progressive disclosure for advanced features
- Focus on mobile-first experience
- Maintain all backend capabilities but expose minimally
